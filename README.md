## 🚀 Quick Start

### Windows, macOS & Linux

1. Extract the ZIP anywhere.

2. Open:

```text
DataViz_Studio.html
```

3. Select a chart type from the left sidebar.

4. Enter your data or click:

```text
⚡ Sample Data
```

5. Click:

```text
▶ Generate Chart
```

6. Export your chart:

```text
⬇ Export PDF
```

## 🔧 Troubleshooting

### Charts Not Displaying

Verify:

- JavaScript is enabled.
- Browser is updated.

---

### 3D Chart Performance Slow

Recommended:

- Use hardware acceleration.
- Close unnecessary applications.

---

### PDF Export Failed

Verify:

- Browser allows downloads.
- Sufficient disk space is available.

---

### Data Appears Incorrect

Verify:

- Labels match values.
- Numbers are entered correctly.
- Data ranges are valid.
